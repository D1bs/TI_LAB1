# Лабораторная работа ТИ №1 — Вариант 10

### 📌 О проекте

Десктопное приложение на Avalonia для шифрования/дешифрования текста двумя методами: **столбцовый метод** с двумя ключевыми словами и **шифр Виженера** с самогенерирующимся ключом.

Работает только с **русскими буквами** (33 буквы, включая Ё). Все остальные символы игнорируются.

---

### 📁 Структура

```
Program.cs              # Точка входа
App.axaml / App.axaml.cs
MainWindow.axaml        # Интерфейс
MainWindow.axaml.cs     # Обработчики, файловые операции
CipherAlgorithms.cs     # ОСНОВНАЯ ЛОГИКА (оба алгоритма)
```

---

### 🔐 Где реализована основная логика

**Весь криптоалгоритм находится в `CipherAlgorithms.cs`:**

| Метод | Описание |
|---|---|
| `DoubleColumnEncrypt(text, key1, key2)` | Шифрование столбцовым методом |
| `DoubleColumnDecrypt(text, key1, key2)` | Дешифрование столбцовым методом |
| `ColumnOrder(key)` | Порядок считывания столбцов по ключу |
| `VigenereAutoEncrypt(text, key)` | Шифрование Виженера |
| `VigenereAutoDecrypt(cipher, key)` | Дешифрование Виженера |
| `FilterRussian(text)` | Фильтрация — только русские буквы |
